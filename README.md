### Игра, в которой есть три сущности:

1. Игровое поле прямоугольной формы
2. Игрок, который может перемещаться по свободным клеткам влево и вправо,
   бегая, если клетка рядом свободна, и прыгая, если занята
3. Деталь - фигура, которую можно поместить в игровое поле или убрать из него

### Суть игры
Выложить детали так, чтобы игровое поле было полностью заполнено,
помещать детали на игрока нельзя
